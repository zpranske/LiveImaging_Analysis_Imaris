LiveImaging\_Analysis\_Imaris

Paradis Lab live imaging analysis pipeline for Imaris surface tracking data



This repository contains the public versions of all scripts and functions needed for the live imaging analysis in Pranske \& Paradis (2025, Elife). This is a custom set of scripts designed for aggregation, post-processing and visualization of Imaris tracking data for synaptic live imaging. simple\_imaris\_crawler.m aggregates raw output files from Imaris, and analyze\_track\_characteristics.m handles main processing. Several smaller scripts are included for specific analyses including k-means clustering, mixed effect model analysis, and new colocalization event analysis. These scripts are publicly available, please contact the authors with any questions.
